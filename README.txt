
Тестовое задание Java

Реализация:

Проект оформлен как артефакт Maven.
Результатом выполнения теста является Yandex.Allure отчёт.

Тестовые файлы:
Директория "src\test\java\operationTest" содержит:

- AdditionOperationTest.java - реализует сценарий для тестирования операции сложения;
- SubtractionOperationTest.java - реализует сценарий для тестирования операции вычитания;
- MultiplicationOperationTest.java - реализует сценарий для тестирования операции умножения;
- DivisionOperationTest.java - реализует сценарий для тестирования операции деления;
- TestBase.java - родительский класс.

Каждое действие (сложение/вычитание/умножение/деление) в отчёте представлено как отдельный тестовый сценарий.
Операнды, операция и результат представлены в отчете как параметры теста.

Вспомогательные файлы:
"src\main\java\processing\DataProcessing.java" - вспомогательный файл для обработки данных.
"src\main\java\enums\OperationTypes.java" - вспомогательный файл, содержащий перечисление операций.
